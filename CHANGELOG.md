# Changelog

## 1.1 - Beta Preview

- Added local encrypted Vault.
- Added credential management, search, folders, details, edit, copy, delete, and restore flows.
- Added current-site matching with subdomain support.
- Added browser popup, full Vault page, and standalone settings page.
- Added content-script autofill and inline login menu.
- Added one-click login, browse-and-fill, and go-to-site actions.
- Added login-save prompt and update-existing-account flow.
- Added password generator with configurable rules.
- Added identity / fill-profile import from Excel, CSV, and `.kpfill`.
- Added selected fill-profile export to `.kpfill`.
- Added selected credential export to RoboForm CSV.
- Added RoboForm / Chrome / Edge / generic CSV credential import.
- Added encrypted Vault backup import and export.
- Added high security mode to disable unlocked Vault and AES-key session caching.
- Added local favicon capture and default icon fallback.
- Added test lab pages for local form-recognition checks.
